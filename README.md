# AI Blog Reviewer


## Run Development Server

1. npm i

   ```text
   npm i
   ```

1. Create `.env.local` file and write a OpenAI API key.

   ```text
   NEXT_PUBLIC_OPENAI_API_KEY="~"
   ```

1. Run a development server

   ```bash
   npm run dev
   ```

1. Open [http://localhost:3000](http://localhost:3000)

## Other Commands

- Lint
  - `npm run lint`
- Format
  - `npm run format`
